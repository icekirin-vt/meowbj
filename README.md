# MEOW OBJ 
* A kitten(smol) sized header only, C99 compliant obj model loader for your graphics programming needs*

## Features:
- Loads vector positions, normals, uv coordinates
- Uses Structure of Arrays(SOA) data format for return
- Easy iteration and interaction
- Easilly hackable for your own needs
- Self dependant! :
    - Custom meowVec3 and meowVec2 formats for intermediary processing and return
    - No extra dependencies
